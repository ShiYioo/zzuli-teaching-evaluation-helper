# ZZULI 教学评价一键满分脚本

**zzuli-teaching-evaluation-helper**

> 郑州轻工业大学教学综合管理平台 - 教学评价自动化工具

## ✨ 特性

- ⭐ **一键满分**: 自动将所有评分项填充为满分（10分）
- 🗑️ **一键清空**: 快速清空所有已填写的内容
- 📝 **自动评语**: 自动填写评语内容
- ⌨️ **快捷键**: 使用 `Ctrl+Q` (Windows/Linux) 或 `Cmd+Q` (Mac) 快速填充
- 🎨 **iOS 设计风格**: 采用 iOS 设计美学，大圆角按钮、SF 字体、流畅动画

## 🚀 安装

### 1. 安装油猴插件

- **Chrome/Edge**: [Chrome 网上应用店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- **Firefox**: [Firefox 附加组件](https://addons.mozilla.org/zh-CN/firefox/addon/tampermonkey/)

### 2. 安装脚本

**从 Greasyfork 安装（推荐）：**

点击访问 → [ZZULI教学评价一键满分](https://greasyfork.org/zh-CN/scripts/561669-zzuli%E6%95%99%E5%AD%A6%E8%AF%84%E4%BB%B7%E4%B8%80%E9%94%AE%E6%BB%A1%E5%88%86)

或手动安装：复制 `教学评价一键满分.user.js` 内容到 Tampermonkey 新建脚本中

## 📖 使用

1. 登录郑州轻工业大学教学综合管理平台
2. 进入教学评价页面，打开任意待评价课程
3. 点击 **⭐ 一键满分** 按钮自动填充所有评分项为满分
4. 检查无误后，点击原有的 **提交** 按钮完成评价
5. 快捷键：`Ctrl+Q` (Windows/Linux) 或 `Cmd+Q` (Mac)

## ⚠️ 注意事项

- 仅供学习交流使用，请根据实际情况认真评价
- 脚本不会自动提交评价，需要手动点击"提交"按钮
- 脚本仅在教学评价页面运行，不收集任何个人信息
